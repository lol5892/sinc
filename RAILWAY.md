# Подключение проекта к Railway (чеклист)

Сделать может **только ты** в браузере: войти в GitHub и Railway. Токен бота **никому в чат не присылай** — только в поля Railway.

## 1. Код на GitHub

1. Репозиторий с папкой этого проекта уже на GitHub (или создай новый и залей код).
2. Убедись, что в репозитории **нет** файла `.env` с секретами (он в `.gitignore`).

## 2. Railway

1. Зайди на https://railway.app и войди (удобно через GitHub).
2. **New project** → **GitHub Repository** → выбери репозиторий **sinc**.
3. Railway сам подхватит настройки из `railway.json`. Если спросит вручную:
   - Build: `npm ci --include=dev && npm run build`
   - Start: `npm run start`

## Если деплой «CRASHED»

Чаще всего в **Variables** на Railway **не добавлен `BOT_TOKEN`** (или неверный токен), либо Railway запустил старую сборку без `railway.json`. В новых версиях кода сервер стартует из скомпилированного `dist-server/` и проверяется через `/api/health` — смотри **Deploy Logs**.

Обязательно добавь: **`BOT_TOKEN`**, **`TELEGRAM_ALLOWED_IDS`**, **`NODE_ENV`** = `production`.

---

## 3. Переменные (Variables)

В сервисе открой **Variables** и добавь:

| Переменная | Значение |
|------------|----------|
| `BOT_TOKEN` | токен от @BotFather |
| `TELEGRAM_ALLOWED_IDS` | два id через запятую |
| `NODE_ENV` | `production` |
| `WEB_APP_URL` | пока **пусто** — заполни после шага 4 |

`PORT` Railway часто подставляет сам — не дублируй, если уже есть.

Сохрани — дождись зелёного деплоя.

## 4. Публичный URL

1. В сервисе: **Settings** → **Networking** → **Generate Domain** (или Public Networking).
2. Скопируй адрес вида `https://xxxx.up.railway.app`.
3. Снова **Variables** → добавь или измени:

   `WEB_APP_URL` = этот же адрес **без** слэша в конце.

4. Подожди пересборку. Открой ссылку в браузере — должен открыться план.

## 5. Данные между перезапусками (по желанию)

Дела хранятся в `data/events.json`. На Railway диск может сбрасываться при деплое. Чтобы не терять план: в проекте Railway добавь **Volume**, смонтируй в `/app/data` (или путь из документации Railway для твоего рантайма).

## 6. Telegram (BotFather)

1. @BotFather → твой бот → настройки **Mini App / Menu Button / домен**.
2. Укажи **тот же** https-домен, что и `WEB_APP_URL`.

## 7. Проверка

1. В боте **/start** → кнопка **«Открыть планер»**.
2. Добавь дело — второму пользователю придёт уведомление в Telegram.
